# Tirana Practice

A practice project for the Build with AI Summer School — data cleaning, a machine learning model, and a Flask web app (with analytics and a map) built on fake Tirana apartment listings.

## Sessions
- Data Lab (v1.0) — clean, explore, analyze, visualize apartment data
- Model Basics (v1.1) — train, save, and compare a price prediction model
- Flask App Fundamentals (v1.2) — a web app displaying the dataset
- Charts & Maps (v1.3) — analytics dashboard with charts and an interactive map
